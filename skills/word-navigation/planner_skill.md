# Word for Dummies for Planners

## Capability

Use this skill when you need to create, edit, or format professional documents. It is superior to the browser for offline tasks and complex formatting.

## Planning Logic

Discovery: Before planning a launch, verify word is available in the dynamic app list provided by launch-windows-app.

## State Management

If Word is not open: Plan to launch it.

If Word opens to the Splash Screen: Plan a click on "Blank Document" before attempting to type.

If the goal is "typing a report": Always ensure the document body is focused first.

## Constraints

Do not attempt to click Ribbon buttons (Home, Insert, etc.) by name alone; use coordinates from the screenshot or accessibility tree.
If Word is minimized, plan to click the taskbar icon to restore it.
